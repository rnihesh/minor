Project Abstract: 
This project focuses on automatic recognition of human emotions from speech signals using deep 
learning techniques. A hybrid CNN-LSTM model is used to extract spatial and temporal features 
from audio signals. Speech features such as MFCCs are extracted and processed using CNN for 
feature learning and LSTM for sequence modeling. The model is trained on the RAVDESS dataset 
to classify emotions like happy, sad, angry, calm, and neutral. The system aims to support mental 
health monitoring by identifying emotional patterns from speech in an automated and non-intrusive 
manner.  
Problem Definition: 
Mental  health  conditions  often  influence  speech  and  emotional  expression.  Manual  emotion 
assessment is subjective and not scalable. The problem is to design an automated system that 
accurately identifies emotions from speech signals using deep learning models for effective mental 
health monitoring.
